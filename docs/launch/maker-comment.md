# Maker first comment

Hi Product Hunt — I built Squid Agent because generating a pretty preview is only half the job. The uncomfortable questions come next: What code did the model actually write? Will it run? What did this attempt cost? Can I take the project somewhere else?

Squid keeps those answers visible. You can inspect every React file, see separate static/runtime/export checks, review the credit receipt, restore earlier versions, and download a portable project with its run and deploy configuration.

For launch, I made the full proof available before signup: open the public example workspace, inspect its original prompt and plan, run the preview, browse the files and quality report, then download the source. If you decide to build, verified accounts get five starter credits—no card required.

I would especially value feedback on the trust layer: what evidence would you need before shipping AI-generated code to production?
