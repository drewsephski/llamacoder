## How to use an example

1. Copy the initial prompt.
2. Replace the audience, domain language, and sample data.
3. Generate the first version and test its primary interaction.
4. Compare the result with the acceptance checklist.
5. Use the focused follow-up prompt only if it matches an issue you observed.

<Callout title="Do not paste every example into one project" type="warn">
  Each example defines a separate product surface. Combining them dilutes the
  primary job and makes the result harder to evaluate.
</Callout>
