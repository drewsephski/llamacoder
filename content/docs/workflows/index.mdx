## Choose the smallest useful loop

For most projects:

1. Generate one coherent primary surface.
2. Test its most important interaction.
3. Repair correctness issues before adding scope.
4. Refine visual hierarchy and responsive behavior.
5. Export only after the preview and quality evidence agree.

This order keeps visual polish from hiding broken behavior.
